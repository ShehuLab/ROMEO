Detailed Description

Implement methods to calculate motions between stable protein states. Based on work by Molloy et al (see BMC Structural Biology (2012) Molloy, Shehu, Elucidating the ensemble of functionally-relevant transitions in protein systems with a robotics-inspired method.

Author
Kevin Molloy, Erion Plaku, Amarda Shehu
Remarks
  • This implementation does not support all the techniques used in the publication above.
