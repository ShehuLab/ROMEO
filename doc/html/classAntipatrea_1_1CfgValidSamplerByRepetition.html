Detailed Description

Repeatedly sample a configuration until a valid one is generated.

Author
Erion Plaku, Amarda Shehu
Remarks
  • This class requires access to CfgSampler for sampling a configuration.
  • It also requires access to CfgAcceptor to test whether the sampled configuration is valid.
  • The user should also set the maximum number of tries to sample a valid configuration by repeatdly calling CfgSampler.
