Detailed Description

Each cell keeps track of the vertices associated with it and the number of times it has been selected for expansion.

Author
Erion Plaku, Amarda Shehu
Remarks
  • A vertex v is associated with a grid cell c iff the projection of the configuration associated with v falls inside the cell c.

inlinevirtual

Get the cell weight.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The weight is high when the cell has few vertices in order to promote expansions from sparsely explored areas.
  • To avoid always selecting the same grid cell, the weight decreases with the number of selections.
