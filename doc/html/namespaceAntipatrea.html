Generate a random subset of r out of n elements.

Author
Erion Plaku, Amarda Shehu
Parameters
nnumber of elements in the array items
itemsarray of elements
rnumber of elements in the subset
Description:
A subset of r elements out of n elements can be selected uniformly at random by selecting the i-th element, i = 0, 1, ..., r - 1, of the r-subset of p as follows:
  • generate uniformly at random an index j in the integer interval [i, n - 1]
  • swap elements p[i] and p[j]
At the end of this procedure, the elements p[0], p[1], ..., p[r - 1] correspond to an ordered r-subset selected uniformly at random from the original n-set p. The elements p[r], ..., p[n - 1] contain the original elements of p that were not selected.

The number of ways of choosing an ordered r-subset out of an n-set p is given by
P(r, n) = n! / (n - r)!.
For example, for r = 2 there are 4! / (4 - 2)! = 12 ordered r-subsets of p = {0, 1, 2, 3}, namely,
{0, 1}, {0, 2}, {0, 3}, {1, 0}, {1, 2}, {1, 3}, {2, 0}, {2, 1}, {2, 3}, {3, 0}, {3, 1}, {3, 2}.
The ordered r-subset is said to be selected uniformly at random when it is selected with probability
1 / P(r, n)
out of all P(r, n) possible ordered r-subsets of the n-set p. Similarly, the number of ways of choosing an unordered r-subset out of an n-set p is given by
C(r, n) = P(r, n) / r! = n! / ((n - r)! r!).
For example, for r = 2 there are 4! / ((4 - 2)! 2!) = 6unordered r-subsets of p = {0, 1, 2, 3}, namely,
{0, 1}, {0, 2}, {0, 3}, {1, 2}, {1, 3}, {2, 3}.
The unordered r-subset is said to be selected uniformly at random when it is selected with probability
1 / C(r, n)
out of all C(r, n) possible unordered r-subsets of the n-set p.

Observe that each unordered r-subset corresponds to r! ordered r-subsets obtained by generating all possible permutations of the elements of the unordered r-subset. Hence, the same procedure that selects uniformly at random an ordered r-subset of the n-set p, can be used to select uniformly at random an unordered r-subset of the n-set p. The unordered r-subset of p is selected when one of the r! corresponding ordered r-subsets is selected. Since an ordered r-subset is selected with probability 1 / P(r, n), then an unordered r-subset is selected with probability
r! (1/ P(r, n)) = 1 / C(r, n).
Remarks
The values of n and r should satisfy the following constraint:
  • 0 <= r <= n

Description:
A permutation A is said to be larger than another permutation B iff there exists an integer j such that
A[0..j-1] = B[0..j-1] and A[j] > B[j],
i.e., it is the same as comparing numbers where the digits of the number form the permutation.

The next permutation of A is defined as the smallest possible permutation B larger than A.

The procedure to generate the next permutation in the lexicographic order is as follows:
  • Starting from right to left examine the array to find the largest tail that is in descending order. Permutation looks like
    p[0] < ... < p[i-2] < p[i-1] < p[i] > p[i+1] > p[i+2] >... > p[n-1].
  • Since we are after the next largest permutation we should keep p[0], p[1],...,p[i-2] unchanged and change p[i-1] to something larger but as small as possible. Thus, we need to find p[k] > p[i-1] among the longest tail which is the smallest possible.
  • Swap p[k] with p[i-1].
  • Reverse tail p[i..n-1].
