Detailed Description

Implement the FELTR algorithm. See IJRR (2010) Shehu, Olson Guiding the Search for Native-like Protein Conformations with an Ab-Initio Tree-based exploration and Olson, Molloy, Shehu In Search of the Protein Native State with a Probabilistic Sampling Approach (JBCC 2011).

Author
Kevin Molloy, Erion Plaku, Amarda Shehu
Remarks
  • This implementation uses Rosetta for the forward kinematics, energy functions, and fragment libraries.

Remarks
  • Function first invokes TreeSamplingBasedPlanner::SetupFromParams(params).
  • It then sets the neighborhood radius (keyword Constants::KW_NeighborhoodRadius)
  • It uses the parameter group associated with the keyword Constants::KW_EST.
  • The parameter values can be specified in a text file as, for example,
    EST { NeighborhoodRadius 1.0 } }
