◆ Vector3_t

typedef double* Antipatrea::Vector3Algebra::Vector3_t

Data type for 3-dimensional vectors.

A 3-dimensional vector v = (a, b, c) is a 3-tuple, where a, b, and c represent the first, second, and third coordinate of v, respectively.

Compute the angle between the vectors v1 and v2.

Parameters
[in]v1a pointer of type Vector3_t representing the first vector.
[in]v2a pointer of type Vector3_t representing the second vector.
Returns
The angle theta between the vectors v1 and v2 defined as
theta = acos(dot(v1, v2) / (norm(v1) * norm(v2)).
The computed angle theta is in the range
[0, Pi].
Warning
If norm(v1) or norm(v2) is close to 0, then 0 is returned as the angle between v1 and v2.
