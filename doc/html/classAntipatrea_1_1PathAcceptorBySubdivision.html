Detailed Description

Test whether or not a path is valid at a given time-step resolution using the subdivision method.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The end configurations (at time = 0 and time = 1) are not checked (assumed to be acceptable).
  • The subdivision method checks the path at different time-step resolutions.
  • It starts at 0.5 and then cuts the resolution in half until it reaches a user-provided minimum time-step resolution.
  • Specifically, the resolution at level i is set to res_i = (1/2)^i.
  • At level i, the path is checked at time steps t_j = 0.5 res_i + j * res_i, starting with j = 0 and continuing as long as t_j < 1.
  • If one of this configurations is found to be not acceptable, then the subdivision method deems the path nonacceptable and returns.
