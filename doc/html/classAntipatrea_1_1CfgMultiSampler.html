Detailed Description

Collection of samplers: select a sampler from the collection with probability proportional to its weight and then use it to sample a configuration.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The collection consists of a number of samplers.
  • Each sampler has a weight associated with it.
  • A configuration is sampled by first selecting a sampler from the collection with probability proportional to its weight.
  • The selected sampler is then used to sample a configuration.
