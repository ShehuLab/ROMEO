Detailed Description

Vertex representation for EST.

Author
Erion Plaku, Amarda Shehu
Remarks
  • In addition to the information stored by PlannerVertex, an ESTVertex also keeps track of the number of neighbors inside a user-specified neighborhood and the number of times the vertex has been selected for expansion.
