Detailed Description

Forward kinematics for a chain in a 2D environment.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The start position of the first link is (0, 0).

virtual void Antipatrea::Chain2D::SetupFromParams ( Params &  params)
virtual

Set the parameter values of the component from the given parameters.

Author
Erion Plaku, Amarda Shehu
Remarks
  • Sets the number of links and the link length (keywords Constants::KW_Chain2D_NrLinks, Constants::KW_Chain2D_LinkLength).
  • It uses the parameter group associated with the keyword Constants::KW_Chain2D.
  • The parameter values can be specified in a text file as, for example,
    Chain2D { NrLinks 8 LinkLength 1.0 }
