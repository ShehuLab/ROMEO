Detailed Description

Representation of the undirected graph (roadmap or tree) constructed by a sampling-based motion planner.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The graph has the following:
    • a set of vertices;
    • a set of edges; and
    • a set of connected components represented as a disjoint set.
  • The set of edges is the union of the edges kept by each vertex. It is kept as a separate set to facilitate enumeration and searching.
  • It is assumed that a sampling-based motion planner will construct an undirected graph, but the edge cost from A to B could be different from the edge cost from B to A. This could happen, for example, when the edge cost is based on the energy change.
  • For this reason, and to facilitate the graph search, the graph maintains the edges as directed edges. In other words, when a sampling-based motion planner makes a connection from vertex A to B, it needs to add two edges to the graph, namely, the edge (A, B) with cost from A to B and the edge (B, A) with cost from B to A.
