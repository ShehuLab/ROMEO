Detailed Description

Motion-planning component which accepts a configuration only if it is within a user-defined distance from a target/goal configuration.

Author
Erion Plaku, Amarda Shehu
Remarks
  • This class requires access to CfgDistance component in order to compute the distance between two configurations.
  • The user should also set the distance threshold.

virtual void Antipatrea::CfgAcceptorBasedOnDistance::SetupFromParams ( Params &  params)
inlinevirtual

Set the parameter values of the component from the given parameters.

Author
Erion Plaku, Amarda Shehu
Remarks
  • Function first invokes CfgAcceptor::SetupFromParams(params).
  • It then sets the distance threshold (keyword Constants::KW_DistanceThreshold).
  • When used as a goal acceptor, it uses the parameter group associated with the keyword Constants::KW_GoalAcceptorBasedOnDistance.
  • When used as general cfg acceptor, it uses the paramater group associated with the keyword Constants::KW_CfgAcceptorBasedOnDistance
  • The parameter value can be specified in a text file as, for example,
    CfgAcceptorBasedOnDistance { DistanceThreshold 0.5 }
    or
    GoalAcceptorBasedOnDistance { DistanceThreshold 0.01 }
