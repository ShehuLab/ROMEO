Detailed Description

Test whether or not a path is valid at a given time-step resolution using the incremental method.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The end configurations (at time = 0 and time = 1) are not checked (assumed to be acceptable).
  • The incremental method checks the path at time steps t_j = (j + 1) tstep, starting with j = 0 and continuing as long as t_j < 1.
  • If one of this configurations is found to be not acceptable, then the incremental method deems the path nonacceptable and returns.
