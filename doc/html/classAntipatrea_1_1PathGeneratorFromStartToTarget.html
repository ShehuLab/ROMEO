Detailed Description

Interface for defining a path that starts at a specific configuration and reaches a specific target configuration.

Author
Erion Plaku, Amarda Shehu
Remarks
  • Classes implementing this interface should ensure that the path starts at the specified configuration (GetFromCfg()) and ends at the specified configuration (GetTargetCfg()).
  • Such class is used by sampling-based motion planners that build a roadmap to connect configurations to one another.
