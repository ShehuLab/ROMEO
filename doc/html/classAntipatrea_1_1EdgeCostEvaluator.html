Detailed Description

Interface for evaluating the direct and reverse cost of a path.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The reverse cost of a path from A to B corresponds to the cost of the reverse path, namely from B to A.
  • Often the direct and reverse costs are the same but not always.
  • This interface allows the user to implement nonsymmetric path-cost evaluators.
