Detailed Description

template<typename Key>
class Antipatrea::Selector< Key >

Given items p1, p2, ..., pn with weights w1, w2, ..., wn, select item pi with probability wi / (w1 + ... + wn)

Author
Erion Plaku
Description:
Items are inserted into a balanced binary tree (according to the weights). This makes it possible to select/update weights in logarithmic time.
