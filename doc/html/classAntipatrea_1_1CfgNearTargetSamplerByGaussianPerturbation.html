Detailed Description

Sample a configuration near a target configuration by using a Gaussian distribution to perturb its values.

Author
Erion Plaku, Amarda Shehu
Remarks
  • This class requires access to CfgManager as it uses the configuration dimension.
  • The user should also set the standard deviation of the Gaussian distribution used for the perturbation of the values of the target configuration.
