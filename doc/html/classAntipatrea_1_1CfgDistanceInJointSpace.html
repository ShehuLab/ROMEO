Detailed Description

Defines the distance between two configurations in joint space.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The distance is computed as the L2 norm of the vector defined by taking the pairwise joint distances as defined by JointSignedDistance component.
  • This class requires access to CfgManager component in order to access the configuration dimension.
  • It also requires access to JointSignedDistance component in order to compute the distance between any two joint values.
