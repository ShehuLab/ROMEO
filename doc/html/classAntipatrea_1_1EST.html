◆ AddVertex()

virtual int Antipatrea::EST::AddVertex ( Cfg *const  cfg)
protectedvirtual

Add a new vertex to the graph with cfg as its associated configuration.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The function also computes the number of vertices that fall inside the neighborhood centered at cfg (configurations whose distance from cfg is no more than GetNeighboorhodRadius(). For each of these configurations, it also increments the number of neighbors counter by one (since cfg will be inside their neighborhoods).
  • The function does not make a copy of cfg; it simply does pointer assignment.
  • The function does not check if cfg is already in the graph. It is the responsibility of the calling function to perform such a check (using FindCfg) if indeed there is a possibility that the same configuration is added to the graph multiple times (which even if it happened would not cause any errors).

Reimplemented from Antipatrea::SamplingBasedPlanner.

◆ GetVertexWeight()

virtual double Antipatrea::EST::GetVertexWeight ( const ESTVertex &  v) const
protectedvirtual

Get the vertex weight.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The weight is high when the vertex has few neighbors inside its neighborhood in order to promote expansions from sparsely explored areas.
  • To avoid always selecting the same vertex, the weight decreases with the number of selections.

Author
