Detailed Description

Defines the distance between two configurations in Euclidean space.

Author
Erion Plaku, Amarda Shehu
Remarks
  • This class treats the configuration values as points in Euclidean space and computes the distance between two configurations as the Euclidean distance in R^dim, where dim is the configuration dimension.
  • This class requires access to CfgManager component in order to access the configuration dimension.
