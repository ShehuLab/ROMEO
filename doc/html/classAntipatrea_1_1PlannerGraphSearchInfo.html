Detailed Description

Wrapper for the planner graph so that it can be used with the graph searching alogrithms, e.g., DFS, BFS, A*, Dijkstra.

Author
Erion Plaku, Amarda Shehu
Remarks
  • The graph-search algorithms in this package work with general, implicit, graph representation.
  • This wrapper takes a PlanerGraph and implements the functions needed by the graph-search algorithms, namely GetOutEdges and IsGoal.
